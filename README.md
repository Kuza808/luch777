# Ресторан "Луч" - Система управления

JavaFX-приложение для автоматизации ресторана.

## Функционал
- Авторизация и регистрация
- Просмотр меню с поиском и фильтрацией
- Корзина и оформление заказа (самовывоз/доставка)
- Бронирование столиков
- Личный кабинет с историей заказов и избранным
- Админ-панель и роль курьера

## Технологии
- Java 17, JavaFX, Supabase, Maven
